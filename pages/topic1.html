Topic 1: Getting Started

This is the content for Topic 1. This page demonstrates how the auto-routing works for static pages.

What is Topic 1?

Topic 1 is an example topic page that shows how easy it is to add new content to your static site. The main index will automatically discover and link to this page.

Key Points About Topic 1

  • This is a static HTML page located in the pages directory
  • The main index.html auto-discovers and routes to this page
  • No framework is needed - this is pure HTML, CSS and JavaScript
  • You can add as many topic pages as you want

How to Add Your Own Topics

Simply create a new HTML file in the pages directory, following the format of this page. The main index will automatically discover it and add it to the navigation.

Note: For the auto-discovery to work in a real production environment, you would need to implement a server-side solution or a build-time process to generate a manifest of available pages.
